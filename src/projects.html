Community Projects
The Ubuntu Program

This is a project by I and Nwagba Okechuchukwu (Developer Student Club UNN Lead) as a means to help bring newbie as well as intermediate developers up to speed and foster collabration across communities.
The Ubuntu Program is a six (6) weeks, virtual peer learning program with the focus on getting learners from Zero to a Hundred on either JavaScript or Python track. The pilot program started with 198 learners, 6 volunteers, 5 team members and 2 sponsors.

Ingressive Weekly

I started a newsletter for the Ingressive community in my local campus and over time it grew reaching audiences from other campuses and beyond. I write majorly about stuff which I think are helpful for newbies and budding developers as well as include new resources every week to help them get better and become world class.
My sources for the newsletter include social media such as twitter, blog posts, main stream newsletter where I curate content that I believe would not be an overkill for upcoming developers
Currently there are about 390 - 400 subscribers to Ingressive Weekly.
